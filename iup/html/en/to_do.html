Windows

  • In Windows Vista the COMPOSITE=YES attribute of the IupDialog is not working as expected. There is still some flicker when the dialog is resized. So we changed the default to NO.
  • IupTabs using native controls in Windows Vista when COMPOSITE=YES works only if MULTILINE=YES, so it is internally overwritten until the problem is solved. (since 3.0)
  • The MDI Window menu is not updated when the child mdi is closed.
