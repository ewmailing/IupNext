IupNormalizeSize

Normalizes all elements natural size to be the biggest natural size among them. All natural width will be set to the biggest width, and all natural height will be set to the biggest height according to is value.

Parameters/Return

void IupNormalizeSize(const char* value, Ihandle *ih_first, ...); [in C]
void IupNormalizeSizev(const char* value, Ihandle **ih_list); [in C]
iup.NormalizeSize{value: string, ih_first, ...: ihandle} [in Lua]

value: normalization direction. Can be HORIZONTAL, VERTICAL or BOTH. These are the same values of the NORMALIZESIZE attribute.
ih_first,... : List of identifiers of interface elements. NULL must be used to define the end of the list in C.
ih_list: NULL terminated list of identifiers of interface elements.

The elements do NOT need to be children of the same parent, do NOT need to be mapped, and do NOT need to be in a complete hierarchy of a dialog.

Notes

Has the same effect of the NORMALIZESIZE attribute of the IupVbox and IupHbox controls, but can be used for elements with different parents and the User size of the elements is changed.
